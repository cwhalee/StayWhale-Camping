## 프로젝트 개요
이 프로젝트는 StayWhale이라는 숙박예약사이트를 진행한 것이며 총 4명의 팀원들이 캠핑 / 커뮤니티게시판 등 테마에 맞춰 나눠 개발했으며 그 중 캠핑 파트를 맡아 구현했습니다.
